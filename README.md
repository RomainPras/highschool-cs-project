## Overview
The goal of this project was to learn the fundamentals of web development. The site covers Aelox Hardware's Aetheris-4 project, a polar kinematics 3D printer which the goal of is to collect funding, to develop a free, open-source 5-axis slicer.

## Tech Stack
* **HTML5**: Structure and content.
* **CSS3**: Styling and layout.
* **Javascript** : Interactivity.
